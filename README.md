# корпус для твитеров DST  

![result](https://github.com/diy-electron1cs/tweeter-cases/blob/main/images/%D0%BF%D1%80%D0%B5%D0%B2%D1%8C%D1%8E.jpg?raw=true)  

## крепление
Твитер крепится на 3 винта M4x10 или на 15. Внутрь корпуса запрессовываются 3 гайки M4.

## рекомендации по печати

### расположение корпуса на столе
Благодаря строению корпуса, деталь печатается без поддержек сферической частью вверх:  

![расположение на столе](https://github.com/diy-electron1cs/tweeter-cases/blob/main/images/%D0%BF%D0%BE%D0%BB%D0%BE%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5_%D0%B4%D0%BB%D1%8F_%D1%81%D0%BB%D0%B0%D0%B9%D1%81%D0%B8%D0%BD%D0%B3%D0%B0.png?raw=true)
### настройки и филамент
- я использовал филамент PLA, подойдут также PETG и ABS
- позиция шва – случайная (Random)
- Fuzzy Skin с параметрами 0.2-0.2
